# CCTV_FFPlay_with_console
CCTV Camera connection control with optional settings and ping tracker
From Command line:
ffplay -rtsp_transport tcp "rtsp://USERNAME:PASSWORD@IP:PORT/chA/B"

A: 1(ch1) , 2(ch2) etc
B:0- mainstream, 1 - sub stream
Requested installation:
ffplay

Python packages:
matpotlib
cv2
